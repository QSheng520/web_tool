# 证件照P图网站

一个功能完善的在线证件照处理工具，基于 HTML + CSS + JavaScript 开发，无需后端服务器即可运行。

## 功能特性

### 📸 图片上传
- 支持点击上传或拖拽上传
- 支持 JPG、PNG 格式
- 上传后实时预览

### 👔 衣服更换
- 8种精心设计的衣服素材（SVG绘制）：
  - 无（不使用衣服）
  - 黑色西装
  - 蓝色西装
  - 灰色西装
  - 白色衬衫
  - 蓝色衬衫
  - 红色西装
  - 蓝色工作服

### 📐 证件照尺寸
- 一寸：295×413px (25×35mm)
- 二寸：413×579px (35×49mm)
- 小二寸：413×531px (35×45mm)
- 护照照：390×567px (33×48mm)
- 身份证照：358×441px (26×32mm)

### 🎛️ 位置调整
- 人物大小缩放（30%-150%）
- 人物上下位置调整
- 衣服大小缩放（50%-150%）

### ✨ 美颜功能
- 美颜：智能磨皮效果
- 美白：提亮肤色
- 瘦脸：面部轮廓调整
- 所有效果支持 0-100 强度调节

### 🎨 背景颜色
- 8种常用证件照背景色：
  - 白色、浅蓝、灰色、红色、绿色、黄色、紫色、橙色

### 💾 导出功能
- PNG 格式（透明背景）
- JPG 格式（标准格式）
- 一键下载到本地

## 使用方法

### 方式一：直接打开
1. 双击 `index.html` 文件
2. 在浏览器中打开即可使用

### 方式二：本地服务器
```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 然后访问 http://localhost:8000
```

## 技术实现

### 前端技术
- **HTML5**: 语义化结构
- **CSS3**: 现代布局（Grid/Flexbox）、CSS变量、动画效果
- **JavaScript ES6+**: 原生Canvas API、文件处理、事件驱动

### Canvas 处理
- 图片绘制与缩放
- 美颜滤镜（像素级操作）
- 衣服素材叠加（SVG to Canvas）
- 图片导出（Canvas to Blob/DataURL）

### 响应式设计
- 桌面端：左侧工具栏 + 中央画布 + 右侧工具栏
- 平板端：堆叠式布局
- 移动端：自适应网格

## 注意事项

1. **图片隐私**：所有图片处理在本地完成，不会上传到任何服务器
2. **建议图片**：上传正面免冠照片效果最佳
3. **导出尺寸**：导出的是画布原始尺寸，可直接用于打印
4. **衣服素材**：当前版本的衣服素材为简化版SVG，实际使用中可能需要根据具体照片调整位置
